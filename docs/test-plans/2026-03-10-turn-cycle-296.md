## End-to-End: Two Complete Rounds

1. Call `start_game(activity_id)`. Verify 2 teams, each with seq=1 (user, has codename) and seq=2 (assistant, has `metadata_json`). Both teams: `current_round=1, round_state="locked"`.
2. Call `publish_all(activity_id)`. Verify all teams: `current_round=2, round_state="drafting", student_summary_text` non-empty, `move_buffer_crdt=None`, `current_deadline > now()`.
3. Write distinct CRDT move buffers to each team (e.g. "Deploy forces to sector 7" and "Fortify position at hill 42").
4. Call `on_deadline_fired(activity_id)`. Verify: all teams `round_state="locked"`; seq=3 user messages contain the respective move text; seq=4 assistant messages exist with growing `metadata_json`.
5. Call `publish_all(activity_id)` again. Verify all teams: `current_round=3, round_state="drafting"`, new `student_summary_text`, `move_buffer_crdt=None`, `current_deadline > now()`.
6. Query all `WargameMessage` for each team. Verify exactly 4 total messages per team (seq 1â€“4), exactly 2 of which are `role="assistant"` (seq=2 and seq=4), no sequence number duplicates.
