# Python 3.14 Debian Packages

Place the following amd64 .deb files from deadsnakes PPA here:

1. `libpython3.14-stdlib_3.14.2-1+noble1_amd64.deb`
2. `libpython3.14_3.14.2-1+noble1_amd64.deb`
3. `python3.14_3.14.2-1+noble1_amd64.deb`
4. `python3.14-venv_3.14.2-1+noble1_amd64.deb`

Download from: https://launchpad.net/~deadsnakes/+archive/ubuntu/ppa/+build/30464571

Install with:
```bash
dpkg -i scripts/deb/*.deb
```
